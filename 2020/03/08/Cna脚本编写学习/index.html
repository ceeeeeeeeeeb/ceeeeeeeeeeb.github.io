0x01 gui需要的函数

dialog

创建一个会话

$1 - 会话标题

$2 - dictionary mapping 默认值

$3 - 回调函数，当调用dbutton_action时，$1为dialog，$2为显示的button名,$3是dictionary mapping参数，是通过drow_text，drow_listener_stage，drow_combobox等函数获取到的用户输入。使用$3[‘key’]即可获取用户输入。

drow_text

在dialog会话中添加用户输入框

$1 - a $dialog object

$2 - the name of this row

$3 - the label for this row

$4 - Optional. The width of this text field (in characters). This value isn’t always honored (it won’t shrink the field, but it will make it wider).

Example

drow_combobox

在dialog会话中添加单选框

Arguments

$1 - a $dialog object

$2 - the name of this row

$3 - the label for this row

$4 - an array of options to choose from

Example

drow_listener_stage

在dialog会话中添加listener选择功能，列出所有的listener。

Arguments

$1 - a $dialog object

$2 - the name of this row

$3 - the label for this row

0x02 回调函数17010_exploit

bcd

当前beacon cd到目录$2

Arguments

$1 - the id for the beacon. This may be an array or a single ID.

$2 - the folder to change to.

Example

bupload

上传文件

Arguments

$1 - the id for the beacon. This may be an array or a single ID.

$2 - the local path to the file to upload

Example

shellcode

生成shellcode

Arguments

$1 - the listener name

$2 - true/false: is this shellcode destined for a remote target?

$3 - x86|x64 - the architecture of the stager output.

Example

sleep binary write

sleep pack函数

＆pack和＆unpack压缩和从字节字符串中提取睡眠类型。 这些函数解释模板中指定的数据。

＆pack接受模板和用逗号分隔的项目列表，以打包成一个字节字符串。 ＆unpack接受一个字节字符串，并根据模板提取项目。

所有读取/写入的默认网络字节(大端序)。 如果脚本选择的话，它们可以指示字节顺序。 模板字符后面的+表示大端序。 字符后面的-表示小端序。

bshell

Ask Beacon to run a command with cmd.exe

Arguments

$1 - the id for the beacon. This may be an array or a single ID.

$2 - the command and arguments to run
