hexo安装踩坑

Posted on 2020-02-23 Edited on 2020-03-02
hexo安装记录

安装Hexo

npm install -g hexo-cli

安装hexo-deployer用于发不到github上

npm install hexo-deployer-git --save

初始化建立项目

hexo init blog

执行后会在当前目录下生成blog文件夹，用于存放标准的Hexo博客项目模板

生成网站

hexo g

执行完毕后，在public目录下可看到我们自己书写的博客文档（.md文件）与所选的博客主题模板链接组合，生成的最终静态网站文件，该目录也差不多就是之后发布到GitHub上的实际文件（实际发布到GitHub的是.deploy_git目录），外部的网站“源代码”不会上传到github.io库

启动本地服务器

hexo s

默认地址是 localhost:4000

主题更换

git clone https://github.com/theme-next/hexo-theme-next themes/next

在网站主题配置文件_config.yml,将theme更换为next，再将themes下的主题配置文件_config.yml中将scheme更改为自己的样式。

github账号关联

使用ssh-keygen生成私钥和公钥

ssh-keygen -t rsa -C "github's maill address"

登录Github，点击头像下的settings，添加ssh,新建一个new ssh key，将id_rsa.pub文件里的内容复制上去。
输入
ssh -T git@github.com测试添加ssh是否成功。

在_config.yml中更改自己的github账号配置,冒号后有空格
